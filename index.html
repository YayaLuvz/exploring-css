Introduction to Tags

Tags come in pairs and between brackets. The following are a list of structural and semantic tags.

Structural

Describing elements and paragraphs

  • html - indicates that this is html code
  • body - indicates itens that should be shown in the main browswer window
  • h - indicates headers (h1, h2, h3, etc)
  • p - indicates paragraph

Semantic

Insert extra information like quotes and citations

  • sub - indicates subscript like this
  • sup - indicates superscript like this like this
  • b or strong - indicates emphasis
  • i or em - indicates italisized
  • br - indicates a line break
  • hr - indicates a horizontal line
  • blockquote or q - indicates a quote (although q in generally avoided)
  • abbr - indicates meaning for abbreviations and acronymns
  • cite - indicates when referencing a oiece of work
  • dfn - indicates the first time use of a word
  • address - indicates contact information is available
  • ins - indicates new text
  • del - idicated deleted text
  • s - indicates text that is so longer relevant

Lists

  • ul - indictaes unordered or bulleted list
  • ol - indicates ordered or numbered list
  • li - indicates list item, must include with ul or ol
